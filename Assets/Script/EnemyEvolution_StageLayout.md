## 進化によるステータス上昇パターン

各系統で統一感のある成長カーブ：

### 浮遊系（A蛾 / G幽体）
HP: 3 → 6 → 10（近接半減で実質倍）
ATK: 1 → 1 → 2
銃がないと辛い。進化するほどライフル必須。

### 突撃系（B赤蟲 / J棘獣の029）
HP: 3 → 5 → 8（低め）
ATK: 1 → 2 → 2（+MultiHit）
間隔: 常に1。毎ターン殴ってくる。ショットガンの的。

### 装甲系（C岩殻 / H甲虫 / J棘獣の030）
HP: 5 → 8 → 12〜14（高い）
ATK: 1 → 1 → 2
間隔: 3 → 2 → 2。遅いが硬い。ライフル推奨。

### 回復系（F苔）
HP: 3 → 7 → 10
ATK: 1 → 1 → 1（攻撃力は低いまま）
SelfBuff で回復するので放置すると永遠に倒せない。
火力で押すか、銃で一気に削る。

### 汚染系（E毛獣022 / G幽体013 / H甲虫018 / I炎植物024 / K王蟲033）
PanelCorrupt 持ちは中盤以降に集中。
盤面に LvUp を撒く → 消すと敵が育つ → さらに撒かれる の悪循環。
早期排除が正解だが、HPが高いものも多い。

## LvUpパネルとの相互作用（進化ライン別）

| 系統 | LvUpで育てた場合の危険度 | 育てる価値 |
|------|------------------------|-----------|
| A.蛾（浮遊） | 高。近接半減×HP増で銃必須に | 中。EXPは美味いが銃弾を消費 |
| B.赤蟲（突撃） | 最高。毎ターン殴るのでHP増=生存ターン増 | 低。リスクに見合わない |
| C.岩殻（装甲） | 中。元から硬いのがさらに硬く | 高。ゆっくりなので安全に稼げる |
| D.多脚 | 中。HP増えるが火力は据え置き | 中。堅実な稼ぎ |
| E.毛獣 | 中〜高。HeavyHitのダメージ増はない | 中。ATK固定なのでHP増分は稼ぎ向き |
| F.苔 | 低。ATK1で回復するだけ | 最高。安全に育てて稼げるカモ |
| G.幽体 | 高。浮遊＋汚染がさらに厄介に | 低。汚染が加速して盤面崩壊 |
| H.甲虫（018蟲巣） | 中。攻撃しないのでHP増は安全 | 高。ただし汚染が増えるリスク |
| I.炎植物 | 高。遠距離なので逃げられない | 低。倒すのに時間かかると被弾増 |
| J.棘獣 | 系統次第 | 通常型は稼ぎ向き、突撃/装甲型は危険 |
| K.王蟲 | 最高。ラスボスを育てるのは自殺行為 | 最高。撃破できればEXP大量 |
