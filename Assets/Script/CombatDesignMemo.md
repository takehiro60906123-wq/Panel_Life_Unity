# Combat Design Memo

## 目的
このメモは、現在の戦闘設計の意図を ZIP 共有時にも伝わるようにまとめたもの。
コードだけでは伝わりにくい以下を記録する。

- 4銃の役割
- 状態異常3種の役割
- 敵タイプごとの推奨銃
- 「有利 / 普通 / 不利」は詰みを作らないための指針
- 中ボス / ボスの状態異常耐性方針

---

## 現在の4銃の役割

### ピストル
- 役割: 安定、保険、危険ターン調整
- 特徴:
  - 低コスト
  - 2連射
  - 命中時に **駆動遅延** を付与する可能性あり
- 向いている相手:
  - 突撃敵
  - 遠距離敵
  - PanelCorrupt 系
- 一言:
  - 「倒す」というより「危険な行動をずらす」銃

### マシンガン
- 役割: 手数、雑魚処理、連射補正
- 特徴:
  - 全ゲージ消費で連射
  - **3発ごとに強化弾 (+1ダメージ)** が入る
- 向いている相手:
  - 雑魚群れ
  - 柔らかい中HP敵
  - MultiHit 系の中量級
- 一言:
  - 「数で押す」「撃てば撃つほど気持ちいい」銃

### ショットガン
- 役割: 迎撃、装甲崩し、前線制圧
- 特徴:
  - 危険敵への迎撃
  - 命中時に **腐食** を付与
  - 同一攻撃中の後続ヒットにも腐食補正が乗る
- 向いている相手:
  - 装甲敵
  - 接近してくる突撃敵
  - ボス / 中ボスの装甲剥がし
- 一言:
  - 「初弾で装甲を崩し、後続弾で食い込む」銃

### ライフル
- 役割: 単体高火力、後衛処理、精鋭処理
- 特徴:
  - 高火力単発
  - まだ特殊効果は未実装
- 向いている相手:
  - 浮遊敵
  - 遠距離敵
  - 装甲敵（ショットガンに次ぐ準最適解）
  - ボス / 中ボス
- 一言:
  - 「確実に抜きたい相手を抜く」銃

## 現在の状態異常3種の役割

### 金縛り (Paralysis)
- 役割: 今すぐ止めたい相手を止める
- 効果:
  - 行動不能
  - 被弾で解除
- 主な用途:
  - 突撃敵
  - 危険な遠距離敵
  - 高威力行動を持つ敵

### 駆動遅延 (Slow)
- 役割: 危険行動を遅らせる
- 効果:
  - 敵: その敵ターンの cooldown 進行をスキップ
  - プレイヤー: ターン終了時に敵の cooldown が追加で進む
- 主な用途:
  - 突撃敵
  - 遠距離敵
  - PanelCorrupt 系
- 補足:
  - 金縛りより軽い妨害
  - 「完全停止」ではなく「時間を作る」効果

### 腐食 (Corrosion / 装甲腐食)
- 役割: 硬い敵を崩す
- 効果:
  - 被ダメージ増加
- 主な用途:
  - 装甲敵
  - ボス / 中ボス
- 補足:
  - ショットガンと相性が良い
  - ただし「無いと詰む」設計にはしない

## 敵タイプ × 銃の相性方針

### 重要方針
- **有利** = かなり楽
- **普通** = 十分戦える
- **不利** = 苦しいが突破可能

**特定の銃が無いと詰む敵は作らない。**
シレン的に気持ちいいのは
「対策があると楽、無くても工夫で突破可能」
のライン。

### 相性表

| 敵タイプ | ピストル | マシンガン | ショットガン | ライフル |
|---|---|---|---|---|
| 突撃敵 | 有利 | 普通 | 有利 | 普通 |
| 装甲敵 | 不利 | 不利 | 有利 | 有利 |
| 遠距離敵 | 有利 | 普通 | 普通 | 有利 |
| 浮遊敵 | 普通 | 普通 | 不利〜普通 | 有利 |
| 雑魚群れ | 普通 | 有利 | 有利 | 普通 |
| ボス / 中ボス | 普通 | 普通 | 有利 | 有利 |

## EnemyType ごとの推奨銃メモ

### Rushing
- 推奨銃:
  - ピストル
  - ショットガン
- 理由:
  - 攻撃が早いため、遅延や迎撃が刺さる

### Armored
- 推奨銃:
  - ショットガン
  - ライフル
- 理由:
  - 腐食と高火力が有効
- 注意:
  - ショットガンが無くてもライフルや時間をかけた処理で突破可能にする

### Ranged
- 推奨銃:
  - ピストル
  - ライフル
- 理由:
  - 危険行動をずらす / 先に抜く価値が高い

### Floating
- 推奨銃:
  - ライフル
- 次点:
  - ピストル
- 理由:
  - 近接しづらい。銃での確実処理が本命

### Normal
- 推奨銃:
  - マシンガン
  - 状況次第でショットガン
- 理由:
  - 手数の価値が出やすい

## AttackPattern ごとの注意

### Normal
- 標準パターン
- 基本の比較対象

### MultiHit
- 危険度が高くなりやすい
- 金縛り / 駆動遅延の価値が上がる
- マシンガンやショットガンの処理テンポ比較にも向く

### HeavyHit
- 危険行動
- 金縛りやライフル先制の価値が高い

### SelfBuff
- 放置すると危険
- ピストル遅延、ライフル処理が有効
- cooldown は 2 寄りにすると駆動遅延が見えやすい

### PanelCorrupt
- 放置危険
- ピストルでずらす / ライフルで落とす 役割を見せやすい

## 中ボス / ボスの状態異常耐性方針

### 金縛り
- 雑魚: 普通に効く
- エリート: 1ターンだけ効く
- 中ボス: 短く効く or 低確率
- ボス: 無効 or 効果半減推奨

理由:
- 完全停止が強すぎるため

### 駆動遅延
- 基本は効いてよい
- 中ボスにも有効でよい
- ボスにも効くが、効きすぎるなら
  - 連続適用制限
  - 1回だけ保証行動
  を検討

### 腐食
- 中ボス / ボスにも有効でよい
- ただし「腐食が無いと削れない」は避ける
- ライフルでも十分戦えるラインにする

## interval 調整メモ
現在の enemyData は interval=1 が多い。
そのままでも動くが、**駆動遅延の気持ちよさが見えにくい** 可能性がある。

優先的に interval=2 を検討する候補:
- 序盤の一部 Normal
- 一部 Armored
- SelfBuff
- PanelCorrupt

方針:
- 全員を遅くするのではなく、
- 「遅延が効いた時に価値が見える敵」を少数混ぜる

## 今後の未実装候補
状態異常は一旦この3種で十分成立している。

現在実装済み:
- 金縛り
- 駆動遅延
- 腐食

未実装候補:
- 睡眠
- 毒
- 目つぶし
- 混乱

優先度:
1. 睡眠
2. 毒
3. 目つぶし
4. 混乱

ただし、しばらくは追加よりバランス確認を優先する。

## 直近の確認ポイント
- ピストル遅延が突撃敵 / 遠距離敵で気持ちよく刺さるか
- ショットガン腐食が装甲敵に対して強すぎないか
- ライフル高火力だけで役割が立っているか
- マシンガン強化弾が埋もれていないか
- 中ボスで金縛りが強すぎないか
- 状態異常アイコンが分かりやすいか

## 一言まとめ
- ピストル = 危険ターン調整
- マシンガン = 数処理と連射補正
- ショットガン = 装甲崩しと迎撃
- ライフル = 精鋭 / 後衛 / 浮遊 / ボス処理

- 金縛り = 今すぐ止める
- 駆動遅延 = 時間を作る
- 腐食 = 硬い敵を崩す

- **有利 / 普通 / 不利 は、詰みを作らないための目安**
